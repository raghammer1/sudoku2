Rules

1. All the grey boxes are pre-filled numbers and the new numbers that you fill won't have any background colour

2. To fill a new number in the sudoku board please choose the number you wish to fill from the below numbers panel, then click on the board where you want to fill

3. If the number is incorrect the box will flash red and the number will disappear plus a MISTAKE will be added

4. If you get 5 mistakes the game will END

5. You also get hints in this game which you can trigger by either pressing H key on your keyboard or by clicking on the on screen hint button

6. Once you have finished the game, you can click on the submit button

7. Reset same game will reset the same game for you but now with only one hint as it is a retry

8. Solve button will give you a possible solution for the present sudoku board

9. And finally the New game button will start a new game for you
